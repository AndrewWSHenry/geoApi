I want to download source code and build the geoAPI

  1. Follow GitHub's guides on setting up Git.
  2. Fork the latest fgpv-vpgf geoAPI repository by following the GitHub forking guide.
  3. Follow the Getting Started of the FGPV geoApi readme
  4. To contribute back, follow the instructions on how to create a pull request.

  3. Report a bug, ask a question or request a feature relating to FGPV geoAPI by filling out a new issue.
